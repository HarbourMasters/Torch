# Manifests

SHA256 hash manifests of reference O2R contents. Used by test scripts
to verify Torch output matches the reference.

Generated by `manifest.sh` from reference O2R files (Shipwright build b48e5f7).

14 unique manifests covering all 17 supported ROMs. NTSC N64 J/U pairs
are identical; PAL MQ Debug dumps (3 ROMs) are identical.
