# O2R Archives

All files gitignored — must be provided locally.

## Directory structure

- **generated.o2r** — Latest O2R generated by test_assets.py
- **reference/** — Reference O2Rs generated by OTRExporter (Shipwright build b48e5f7)

## Reference O2R naming

`{version}_{sha}.o2r` where `{sha}` is the first 6 chars of the ROM SHA1.

Example: `pal_gc_0227d7.o2r`
