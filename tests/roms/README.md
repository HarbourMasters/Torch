# Integration Test ROMs

Place ROM files here for integration testing. This directory is gitignored.

## Supported ROMs

| Game | Version | Expected Filename | SHA-1 |
|------|---------|-------------------|-------|
| Super Mario 64 | US | `sm64.us.z64` | `9bef1128717f958171a4afac3ed78ee2bb4e86ce` |

Tests that require a ROM will be skipped (not failed) when the ROM is not present.
